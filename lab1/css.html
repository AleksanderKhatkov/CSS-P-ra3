Блоки

В мире Веб-дизайна существуют тысячи способов решения одной и тоц же проблемы. Причём их количество растёт с каждым днём.

Вместо одного правельного способа решения задачи, к счастью или к несчастью, предоставляется выбор из тысячи вариантов. Возможно, именно богатый набор вариантов делает дизайна таким увлекателным и в то же время ошеломляющим.

CSS предоставленного способа решения типичных задач веб-дизайна.
